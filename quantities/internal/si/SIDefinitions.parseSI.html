    Function SIDefinitions.parseSI

    Parses a statically-typed Quantity from a string at run time.

    Q parseSI(Q, S) (
      S str
    )
    if (isQuantity!Q && isSomeString!S);

    Throws a DimensionException if the parsed quantity doesn't have the same dimensions as Q.

    Parameters

    NameDescription
    Q the type of the returned quantity.
    str the string to parse.

    Example

    alias Time = typeof(second);
Time t = parseSI!Time("90 min");
assert(t == 90 * minute);
t = parseSI!Time("h");
assert(t == 1 * hour);
