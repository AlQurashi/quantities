    Function SIDefinitions.siFormat

    Formats a SI quantity according to a format string known at run-time.

    S siFormat(S, Q) (
      S format,
      Q quantity
    )
    if (isSomeString!S && isQVariantOrQuantity!Q);

    Parameters

    NameDescription
    format The format string. Must start with a format specification for the value of the quantity (a numeric type), that must be followed by the symbol of a SI unit.
    quantity The quantity that must be formatted.

    Example

    QVariant!double speed = 12.5 * kilo(meter) / hour;
assert("%.2f m/s".siFormat(speed) == "3.47 m/s");
